
# [ Bash Tips ] #

## 1. 마지막 working directory로 전환 ##

`cd -`


## 2. multiple command 실행 ##

`cmd1; cmd2 ; cmd3`

- 그런데, cmd1의 Successful이 요구될 때, 

`cmd1 && cmd2`




## 3. 과거 실행 명령어 검색 ##

`CTRL+r <검색어>`



## 4. Linux Terminal unfreeze ##

- 실수로 `CTRL +S` 를 눌렀을 때, ==> shell이 Freezing됨

- Freezing해제를 위해 => `CTRL+Q`


## 5. 쉘터미널에서 첫칸, 마지막칸 이동 ##
- 첫칸 : CTRL + A
- 마지막칸 : CTRL + E
